- direct_rules.txt: 直连 ipset DIRECT
- direct_extra.txt: 直连 ipset DIRECT
- passwall_rules.txt: VPN ipset PASSWALL
- passwall_rules.txt: VPN ipset PASSWALL
- dns_rules.txt: 域名规则，生成 /etc/dnsmasq.d/50-rules.conf
- dns_extra.txt: 域名规则，生成 /etc/dnsmasq.d/50-rules.conf
